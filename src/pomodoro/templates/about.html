About

Pomodoro Timer Project

By: Krish Shah and Bhakti Ramani

Course: ECEN 5713 - Advanced Embedded Systems Development (AESD),
University of Colorado Boulder

This project implements a Pomodoro Timer designed to help users manage work and break sessions efficiently using the popular Pomodoro technique. The timer allows users to start, pause, and reset their work sessions, with different audio tones for each event to provide clear feedback.

The Pomodoro Timer is built to run on a Raspberry Pi environment using a custom Buildroot Linux image. It interacts with hardware components like an LCD display and a buzzer to give real-time updates and alerts to the user.

You can find the complete project repository here: GitHub Repository Checkout Overview Wiki page for further information.
